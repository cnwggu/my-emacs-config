ReflectionMethod::export

(PHP 5)

ReflectionMethod::export — Export a reflection method.

说明

public static string ReflectionMethod::export ( string $class , string $name [, bool $return = false ] )

Exports a ReflectionMethod.

Warning

本函数还未编写文档，仅有参数列表。

参数

class

The class name.

name

The name of the method.

return

设为 TRUE 时返回导出结果，设为 FALSE（默认值）则忽略返回。

返回值

如果参数 return 设为 TRUE，导出结果将作为 string 返回，否则返回 NULL。

参见

  • ReflectionMethod::__construct() - Constructs a ReflectionMethod
  • ReflectionMethod::__toString() - Returns the string representation of the Reflection method object.
